# Memory Recall Skill

当用户的需求可能跟历史有关，或疑似需要回忆过往上下文时，加载本 Skill。

## 使用方式

运行检索脚本获取相关记忆：

```bash
python -X utf8 ~/.claude/skills/recall/recall.py "检索关键词"
```

脚本会执行混合检索（向量 70% + BM25 30%），返回 top-15 相关记忆。

## 流程

1. 从用户当前消息中提炼检索关键词（可以是多个短语）
2. 调用 `recall.py` 执行混合检索
3. 阅读返回的记忆，整合到你的回答中
4. 如果记忆中有成熟直觉（confidence ≥ 0.75），告知用户并建议使用 `self-evolve` Skill

## 注意

- 不需要每次对话都检索，只在相关时使用
- 检索关键词尽量具体，避免过于宽泛
- 如果第一次检索结果不理想，可以换关键词再试
